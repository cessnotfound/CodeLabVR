# CodeLabVR

CodeLab VR

CodeLab VR is a virtual reality-based learning system developed to help students understand fundamental programming concepts through interactive and immersive learning experiences. The system uses culturally contextualized scenarios, such as Filipino everyday environments, to make programming lessons more engaging, relatable, and easier to understand.

This project was developed in partial fulfillment of the requirements for the Capstone Project of the Bachelor of Science in Information Technology (BSIT) program under the College of Computing and Information Sciences (CCIS) at Mariano Marcos State University (MMSU).
